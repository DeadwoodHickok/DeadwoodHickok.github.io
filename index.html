Star Wars i3lock

I was getting tired of the default (and very ugly) lock screens so I thought I'd try my hand at editing the open source i3Lock fancy project to allow it to show a topical Star Wars quote, as well as the Empire or Jedi symbols depending on average screen colour.

In the wise words of master Yoda, "Do or do not, there is no try." And so I did. With a little time and a effort, and some collaboration with a peer of mine with whom I share a love of code and pop culture, we managed to build the lockscreen you see before you, and the very one I use to this day.

Circular Startpage

Whilst browsing GitHub one day, I stumbled across Fuyuneko's Startpage (to see more, click here). With a minimalistic design and expandble elements it immediately caught my eye. I forked the repository and started to tweak and tinker with it.

The simple project that I originally merely intended to be a few small CSS changes to adjust the aesthetics, soon became a fully blown new idea. Based off the groundworks of the expanding blocks, I aimed to create a startpage in the same vein, however with a circular, almost bubble like feel. I eventually achieve my goal, after learning much more about Javascript than I had ever intended, and I eventually hosted the site, which you can view here
